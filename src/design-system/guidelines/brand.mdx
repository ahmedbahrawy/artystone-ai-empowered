# Arty Stone Clinic Brand Guidelines

## Brand Overview
Arty Stone Clinic represents the perfect blend of modern healthcare and timeless elegance. Our brand identity combines professionalism, warmth, and sophistication to create a welcoming and trustworthy healthcare environment.

## Colors

### Primary Colors
- Primary Blue (`#2E90FF`): Our main brand color, representing trust, professionalism, and clarity
- Secondary Earth (`#978368`): Our accent color, bringing warmth and natural elegance

### Neutral Palette
- Light: `#F9FAFB` to `#111827`
- Used for text, backgrounds, and UI elements
- Provides balance and readability

### Semantic Colors
- Success: `#10B981` - Positive actions and confirmations
- Warning: `#F59E0B` - Cautionary messages
- Error: `#EF4444` - Critical alerts
- Info: `#3B82F6` - Informational messages

## Typography

### Font Families
- Sans-serif: Inter
  - Primary font for UI elements and body text
  - Clean, modern, and highly readable
- Serif: Playfair Display
  - Used for headings and featured text
  - Adds elegance and sophistication
- Monospace: JetBrains Mono
  - Used for technical content and code
  - Ensures clarity in technical documentation

### Type Scale
- xs: 0.75rem
- sm: 0.875rem
- base: 1rem
- lg: 1.125rem
- xl: 1.25rem
- 2xl-9xl: 1.5rem to 8rem

### Font Weights
- Regular (400): Body text
- Medium (500): Emphasis
- Semibold (600): Subheadings
- Bold (700): Headings
- Other weights available for specific needs

## Spacing

### Scale
- Base unit: 0.25rem (4px)
- Consistent spacing scale from 0.125rem to 24rem
- Used for margins, padding, and layout gaps

### Layout
- Container widths: sm to 2xl
- Responsive breakpoints:
  - sm: 640px
  - md: 768px
  - lg: 1024px
  - xl: 1280px
  - 2xl: 1536px

## Components

### Buttons
- Primary: Solid background with white text
- Secondary: Border with transparent background
- Consistent padding and rounded corners
- Clear hover and active states

### Cards
- Subtle shadows for depth
- Consistent border radius
- Light backgrounds with dark text
- Optional hover animations

### Forms
- Clear input states (default, focus, error)
- Consistent spacing between elements
- Helpful validation messages
- Accessible labels and placeholders

## Animation

### Principles
- Subtle and purposeful
- Enhances user experience without distraction
- Consistent timing and easing

### Variants
- fadeIn: Smooth opacity transitions
- slideInFromBottom: Gentle upward movement
- slideInFromRight: Horizontal entrance
- scale: Size adjustments
- stagger: Coordinated group animations

## Accessibility

### Color Contrast
- Minimum contrast ratio of 4.5:1 for text
- Clear distinction between interactive elements
- Sufficient contrast in both light and dark modes

### Typography
- Minimum text size of 16px for body content
- Clear hierarchy in headings
- Adequate line height for readability

### Interactive Elements
- Clear focus states
- Sufficient touch targets (minimum 44x44px)
- Keyboard navigation support

## Dark Mode

### Colors
- Inverted color scheme with careful consideration for eye strain
- Maintained contrast ratios
- Subtle gradients and overlays

### Components
- Adjusted shadows and borders
- Maintained readability
- Consistent interactive states
